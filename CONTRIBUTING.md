# Contributing

Thanks for taking the time to contribute.

## Ways to Contribute

- Report bugs with clear repro steps and expected vs. actual behavior.
- Suggest improvements to UX, accessibility, or performance.
- Submit focused pull requests with tests when applicable.

## Development Setup

- Use the instructions in README to load the extension in Chrome or Firefox.
- Run `npm test` to execute unit tests.

## Pull Request Guidelines

- Keep changes small and focused on a single concern.
- Update documentation when behavior or UI changes.
- Run `npm test` and include results in the PR description.
- Avoid adding new dependencies without discussion.

## Reporting Issues

Include the following when filing a bug:

- Browser name and version
- Extension version
- Reproduction steps
- Expected behavior
- Actual behavior
- Screenshots or console logs when relevant
